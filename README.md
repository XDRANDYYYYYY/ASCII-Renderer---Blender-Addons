# ASCII Renderer — Blender Addons
## About

An add-on that convert 3d scene in Blender to ASCII art.

## Installation

Download the zip files and follow the instructions here: https://github.com/Pullusb/How_to_install_Blender_addons

## How to use

After you import the addon, it should appear here

<img width="327" height="407" alt="Screenshot 2026-07-19 at 1 23 25 AM" src="https://github.com/user-attachments/assets/a21db502-13c9-4999-a893-ccfa6b44172e" />

Click it, make sure you have an active camera in your scene, place the camera towards the object, set the height and width you want, and click render

Open the text editor in Blender and choose the exported "ascii_output.txt", and you should see the result.
